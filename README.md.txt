En vez de Bootstrap utilicé CSS Grid ya que ya viene integrado en CSS. Además de esto, CSS grid realiza una parte del ajuste de la pantalla por sí solo.
Fue necesario especificar el tamaño y el comportamiento de una pantalla menor de 700px, pero una pantalla de mayor tamaño se ajusta automaticamente por CSS Grid.
El motivo principal por el que elegí este framework es porque es muy veloz a la hora de desarrollar y porque se acopla muy bien con las otras funciones y los otros
frameworks que tiene CSS tales como flexbox.
